# PRD 17: Templates
> Phase: 2 | Priority: P1
> Status: Draft | Date: 2026-03-03

---

## Problem Statement

New users shouldn't start from a blank slate. Templates accelerate onboarding by providing pre-configured workspaces, views, and scoring frameworks based on industry best practices.

---

## Goals

1. Library of ready-to-use workspace templates
2. Templates cover roadmapping, prioritization, and feedback workflows
3. Templates are fully editable after application
4. Users can create and share their own templates

---

## Detailed Requirements

### 1. Template Library

**Workspace Templates (full workspace setup):**
- Product Management Essentials — basic roadmap + prioritization
- Full Product Management Workflow — roadmap + prioritization + feedback + OKRs
- Agile Sprint Planning — sprint-based with story points
- Technology Roadmap — infra/platform planning
- Marketing Campaign Planner — campaign timeline
- Design System Tracker — component tracking

**View Templates:**
- Product Roadmap (Timeline)
- Now/Next/Later (Board)
- Release Plan (Gantt)
- Value vs Effort (Chart)
- RICE Scoring (Table)
- Feature Backlog (Table with priority sorting)

**Prioritization Templates:**
- RICE Scoring Framework
- Weighted Scoring (customizable)
- MoSCoW Framework
- Kano Model
- Value vs Effort Matrix
- ICE Scoring (Impact, Confidence, Ease)

### 2. Template Application
- Preview template before applying
- Apply to new workspace or existing
- All elements fully editable after application
- Template includes: item types, custom fields, statuses, views, sample data (optional)

### 3. Custom Templates
- Save current workspace configuration as template
- Share templates within organization
- Community templates (future: marketplace)

---

## Success Criteria
- [ ] 10+ workspace templates available
- [ ] 6+ view templates
- [ ] 5+ prioritization framework templates
- [ ] Template preview before applying
- [ ] Save custom templates from existing workspace
